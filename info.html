Methodology

This project was originally documented in LegendKeeper and later exported as static HTML.

Data sources

  • Suburb centroids and CBD distance from internal dataset
  • Public transport and driving commute times collected per suburb

Processing

  • Commute times averaged per suburb and transport mode
  • Rows without public transport commute data excluded
  • Suburbs filtered to CBD distance ≤ 20 km for charts

Visualisation

  • Charts rendered client-side using Chart.js
  • Line graphs ordered by CBD distance ascending
  • Hover tooltips show suburb names and values
